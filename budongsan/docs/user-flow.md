🏡 나만의 스마트 부동산 시트 웹/앱 서비스 사용자 흐름 (User Flow) 🏡
본 문서는 '나만의 스마트 부동산 시트' 웹/앱 서비스의 주요 사용자 흐름을 정의합니다. 각 기능별로 사용자가 어떤 화면을 거쳐 목표를 달성하는지 시각적으로 이해하기 쉽게 명세하며, 이전 화면 설계서의 화면 ID를 참조합니다.

1. 개요
사용자 흐름 문서는 사용자가 서비스 내에서 경험하게 될 여정을 시나리오 기반으로 설명합니다. 이는 개발팀이 기능 간의 연결성을 이해하고, 사용자 경험을 최적화하는 데 중요한 가이드라인이 됩니다.

2. 주요 사용자 흐름 상세
2.1. 흐름 1: 회원가입 및 로그인 (Sign-up & Login)
목적: 사용자가 소셜 계정을 통해 서비스에 회원가입하고 로그인하여 접근합니다.

시작점: 서비스 접속

경로:

서비스 접속

사용자가 웹 브라우저 또는 앱을 통해 서비스에 접속합니다.

시스템: 로그인 상태를 확인합니다.

시작/로그인 페이지 (LOGIN_001)

서비스 로고, 슬로건, 간단한 소개 문구가 표시됩니다.

사용자 액션: "카카오로 시작하기", "네이버로 시작하기", "구글로 시작하기" 중 하나를 클릭/탭합니다.

소셜 서비스 인증 페이지 (외부)

선택한 소셜 서비스의 로그인 및 동의 요청 페이지로 리디렉션됩니다.

사용자 액션: 소셜 계정으로 로그인하고 서비스 연동에 동의합니다.

예외: 소셜 서비스 로그인 실패 또는 동의 거부 시, LOGIN_001 페이지로 돌아오며 "소셜 로그인에 실패했습니다. 다시 시도해주세요." 또는 "서비스 연동에 동의해야 이용할 수 있습니다."와 같은 오류 메시지가 표시됩니다.

메인 검색 페이지 (SEARCH_001)

인증 성공 시 서비스에 자동 로그인되며, 메인 검색 페이지로 이동합니다. (최초 로그인 시 회원가입 처리)

시스템: 로그인된 사용자 정보를 기반으로 개인화된 설정(예: 면적 단위)을 로드합니다.

예외: 백엔드에서 사용자 정보 처리 중 오류 발생 시, LOGIN_001 페이지로 돌아와 "서비스 로그인 처리 중 오류가 발생했습니다. 잠시 후 다시 시도해주세요." 메시지가 표시됩니다.

2.2. 흐름 2: 매물 검색 및 단지 상세 확인 (Property Search & Detail)
목적: 사용자가 특정 지역 또는 주소의 단지를 검색하고 상세 정보를 확인합니다.

시작점: 메인 검색 페이지 (SEARCH_001) (로그인 후)

경로:

메인 검색 페이지 (SEARCH_001)

사용자 액션 (옵션 1: 지역 기반): '광역시/도' 드롭다운 선택 -> '시/군/구' 드롭다운 선택.

사용자 액션 (옵션 2: 주소 입력): 검색 입력창에 주소/단지명 입력 -> 검색 버튼 클릭/탭.

시스템: 선택/입력된 조건에 따라 단지 목록을 백엔드에 요청하고 로딩 상태를 표시합니다.

사용자 액션 (필터링): 가격대, 면적대, 준공년도 등 필터 조건을 설정합니다. (필터 적용 시 검색 결과 테이블 실시간 업데이트)

검색 결과 테이블 (SEARCH_001 내)

검색 조건에 맞는 단지 목록이 테이블 형태로 표시됩니다.

시스템: 검색 결과가 없을 경우, "검색 결과가 없습니다. 다른 조건으로 검색해보세요." 메시지를 표시합니다.

사용자 액션 (옵션): 특정 단지 행에 마우스를 올리거나 클릭/탭하여 단지 미리보기 팝업/패널을 확인합니다. (이때 '자세히 보기' 버튼을 통해 DETAIL_001로 이동 가능)

사용자 액션: 관심 있는 단지 행을 클릭/탭합니다.

단지 상세 정보 페이지 (DETAIL_001)

선택한 단지의 상세 정보(단지 정보, 면적별 정보)가 표시됩니다.

시스템: 단지 상세 정보 로딩 중에는 로딩 스피너를 표시합니다.

사용자 액션: '단지 내 면적별 정보' 테이블에서 관심 있는 면적 행 옆의 '시트에 추가' 버튼을 클릭/탭합니다.

예외: 이미 시트에 추가된 매물인 경우, '시트에 추가' 버튼이 "시트에 추가됨"으로 변경되거나 비활성화됩니다.

나만의 관심 매물 시트 (MY_SHEET_001)

선택된 면적 정보가 사용자의 '나만의 관심 매물 시트'에 자동으로 추가됩니다. (추가 성공 토스트 메시지 "시트에 추가되었습니다!" 표시)

시스템: 시트 데이터가 실시간으로 업데이트됩니다.

2.3. 흐름 3: 나만의 시트 관리 및 매물 비교 (Manage My Sheet & Compare Properties)
목적: 사용자가 자신의 시트를 관리하고, 시트 내 매물들을 비교 분석합니다.

시작점: 나만의 관심 매물 시트 페이지 (MY_SHEET_001)

경로:

나만의 관심 매물 시트 페이지 (MY_SHEET_001)

사용자가 추가한 매물 목록이 엑셀 스타일 테이블로 표시됩니다.

시스템: 시트 데이터 로딩 중에는 로딩 스피너를 표시하고, 시트가 비어있을 경우 안내 메시지를 표시합니다.

사용자 액션 (옵션): '나만의 메모', 사용자 정의 컬럼, 태그 필드를 인라인으로 편집합니다. (편집 시 자동 저장 또는 저장 버튼 활성화)

사용자 액션 (옵션): '컬럼 추가' 버튼을 클릭/탭하여 새로운 사용자 정의 컬럼을 추가합니다. (컬럼 이름 입력 팝업 표시)

사용자 액션 (옵션): 필터/정렬 컨트롤을 사용하여 매물 목록을 탐색합니다.

사용자 액션: 비교하고 싶은 2개 이상의 매물 행의 체크박스를 선택합니다. (선택 수에 따라 '비교하기' 버튼 활성화/비활성화)

매물 비교 분석 리포트 팝업/모달 (COMPARE_001)

사용자 액션: 하단의 '비교하기' 버튼을 클릭/탭합니다.

선택된 매물들 간의 핵심 비교 분석 리포트가 팝업/모달로 표시됩니다.

시스템: 비교 분석 로딩 중에는 로딩 스피너를 표시합니다.

예외: 비교에 필요한 데이터가 부족하거나 유의미한 비교가 어려운 경우, "비교에 필요한 데이터가 부족합니다." 또는 "선택된 매물 간의 유의미한 차이를 발견하기 어렵습니다."와 같은 안내 메시지가 표시됩니다.

사용자 액션: 리포트를 확인 후 닫기 버튼을 클릭/탭합니다.

나만의 관심 매물 시트 페이지 (MY_SHEET_001)

다시 시트 페이지로 돌아옵니다.

사용자 액션 (옵션): '선택 매물 삭제' 버튼을 클릭/탭하여 선택된 매물을 삭제합니다. (삭제 확인 팝업 표시 후 진행)

사용자 액션 (옵션): '시트 내보내기' 버튼을 클릭/탭하여 시트 데이터를 CSV 또는 Excel 파일로 다운로드합니다. (다운로드 포맷 선택 옵션 제공)

2.4. 흐름 4: 개별 매물 상세 분석 (Individual Property Detailed Analysis)
목적: 사용자가 '나만의 시트' 내 특정 매물에 대한 심층적인 정보와 시세 변동 추이를 확인합니다.

시작점: 나만의 관심 매물 시트 페이지 (MY_SHEET_001)

경로:

나만의 관심 매물 시트 페이지 (MY_SHEET_001)

사용자 액션: 매물 리스트 테이블에서 특정 매물 행을 클릭/탭합니다.

개별 매물 상세 분석 팝업/패널 (ANALYSIS_001)

선택한 매물의 핵심 요약, 상세 시세 변동 추이(테이블 및 그래프), 중요 고지 문구가 표시됩니다.

시스템: 상세 분석 데이터 로딩 중에는 로딩 스피너를 표시합니다. 시세 정보가 부족할 경우 "시세 정보가 부족합니다." 메시지를 표시합니다.

사용자 액션 (옵션): 시세 변동 추이 그래프의 기간 선택 옵션(1년, 3년, 5년, 전체)을 변경합니다. (그래프 및 테이블 데이터 실시간 업데이트)

사용자 액션: 내용을 확인 후 닫기 버튼을 클릭/탭합니다.

나만의 관심 매물 시트 페이지 (MY_SHEET_001)

다시 시트 페이지로 돌아옵니다.

2.5. 흐름 5: 알림 설정 (Notification Settings)
목적: 사용자가 관심 부동산 정보의 변화에 대한 알림 설정을 관리합니다.

시작점: 모든 주요 페이지의 상단 헤더 (예: 메인 검색 페이지 SEARCH_001, 나만의 시트 페이지 MY_SHEET_001)

경로:

현재 페이지 (예: SEARCH_001)

사용자 액션: 상단 헤더의 알림 아이콘을 클릭/탭합니다.

알림 설정 페이지 (NOTI_001)

알림 종류별(가격 변동, 신규 실거래가, 맞춤 조건) 토글 스위치 및 세부 설정 필드가 표시됩니다.

알림 방식(웹 푸시, 앱 푸시) 선택 체크박스가 표시됩니다.

시스템: 현재 저장된 알림 설정 상태를 불러와 표시합니다.

사용자 액션: 원하는 알림 설정을 변경하고, '설정 저장' 버튼을 클릭/탭합니다.

예외: 푸시 알림 권한이 없는 경우, 권한 요청 팝업 또는 설정 가이드 링크를 제공합니다.

현재 페이지 (예: SEARCH_001)

설정 저장 성공 토스트 메시지("알림 설정이 저장되었습니다!")가 표시되며, 이전 페이지로 돌아옵니다.

예외: 설정 저장 실패 시 오류 메시지가 표시됩니다.

2.6. 흐름 6: 마이페이지 접근 및 관리 (Access My Page & Manage)
목적: 사용자가 자신의 계정 정보를 확인하고, 필요한 경우 회원 탈퇴를 진행합니다.

시작점: 모든 주요 페이지의 상단 헤더 (예: 메인 검색 페이지 SEARCH_001, 나만의 시트 페이지 MY_SHEET_001)

경로:

현재 페이지 (예: SEARCH_001)

사용자 액션: 상단 헤더의 사용자 프로필 아이콘/닉네임을 클릭/탭합니다.

마이페이지 (MYPAGE_001)

연동된 소셜 계정 정보가 표시됩니다.

사용자 액션 (옵션): '회원 탈퇴' 버튼을 클릭/탭합니다.

회원 탈퇴 확인 팝업/모달

회원 탈퇴에 대한 최종 확인 메시지("정말 탈퇴하시겠습니까? 모든 개인 데이터가 삭제됩니다.")가 표시됩니다.

사용자 액션: '확인' 또는 '취소'를 클릭/탭합니다.

시작/로그인 페이지 (LOGIN_001)

회원 탈퇴 성공 시, 서비스에서 로그아웃되며 LOGIN_001 페이지로 이동합니다. (탈퇴 성공 메시지 표시)

예외: 탈퇴 취소 시 MYPAGE_001로 돌아옵니다.

예외: 탈퇴 처리 중 오류 발생 시, MYPAGE_001에서 오류 메시지가 표시됩니다.
